Kotlin Compose Multi-Platform

A multi-platform UI framework for Kotlin, based on Jetpack Compose

Compose Multi-Platform

Compose Multi-Platform is a framework for creating UIs that work across multiple platforms including Windows, MacOS, Linux, Android and iOS.

Compose MP uses Kotlin and is based on Jetpack Compose. The library is developed by JetBrains and the open-source community.
